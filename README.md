# Kobeni Wallpaper
check it out on the [Steam Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=3604242511)! also drop a rating ;)

## Features
1. Puppet Warp animations by me!
2. FX including smoke trail, floating ash, god rays, CRT & VHS effects
3. Audio visualizer, audio responsive visualizer
4. Media integration / media detection (song title, artist, and album cover).
5. Customizable Date/time visuals
6. Basic intro & hover interactive animations
7. Far away and close up styles for desktop icon space
8. Other Customizable user properties with html styling specified [here](https://github.com/Iemontine/Kobeni-Wallpaper/blob/47d114a947567daea386660cad0f68e778734531/project.json)
9. Basic parallax settings

## Getting Started
1. Clone this directory in ```path_to\Steam\steamapps\common\wallpaper_engine\projects\myprojects\```
2. Open in Wallpaper Engine in Wallpaper Editor > View All Projects > THEATER KOBENI
